What This Tool Is For

Start from an address, choose a radius, pull nearby points of interest, and build a reusable scouting board for drone planning. Scouts can save locations, mark what is photographed, attach notes or media, and reopen the same territory later with the same context intact.

The product is intentionally radius-first. The key question is not just "what exists here?" but "what exists within 1, 5, 10, 25, 50, or 100 miles of the area I care about?"

Core Workflow

1
Create or select a scout.

Profiles are lightweight workspaces for each operator, consultant, or teammate.

2
Search from an address.

Pick a radius band, select categories, and run the nearby search.

3
Review map and results.

Use the clustered map, synchronized list, and coverage ladder to understand the area quickly.

4
Queue, capture, and revisit.

Save points, attach media, export coordinates, and reopen the same scout later.

Operational Notes

  • Fresh sessions may show a short loading state before the Mapbox map and provider counters finish initializing.
  • The `50 mi` and `100 mi` radius bands remain beta intentionally.
  • `100 mi (beta)` is still best used one category at a time.
  • Very dense downtown searches are still faster to triage from the synchronized result list before saving queue pins.
  • Saved-scout reload now restores the live map to the scout's working area instead of falling back to the default center.
  • CSV and GeoJSON exports remain available whenever a scout profile is selected.
  • The queue is the durable work surface; replaying a saved search is how you reopen a prior planning pass.

Recommended Use Pattern

  • Create one scout profile per operator, consultant, or campaign.
  • Use smaller radii first to understand local density before moving to wider bands.
  • Queue only the locations worth follow-up so exports stay high-signal.
  • Attach media or notes as soon as field knowledge is gathered, not later from memory.
  • Watch the provider guardrails during live exploration so you can preserve budget for real fieldwork.
